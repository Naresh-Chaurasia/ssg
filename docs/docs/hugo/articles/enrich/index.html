    Content enrichment

    You can enrich your content by using shortcodes or inserting HTML into the Markdown file.

    Shortcodes

    Hugo offers a number of built-in shortcodes. Themes usually add additional shortcodes you can use. You can also create your own.

    Shortcodes are processed when the Markdown sources are parsed into HTML.

    A full list of the built-in shortcodes is available in the Hugo documentation.

    The shortcodes added by a theme are usually described in the theme’s README.md file.

    Using shortcodes

    1. Open a Markdown file.
    2. Insert a shortcode.

    Creating shortcodes

    You can create your own shortcodes. This procedure presents an example of creating a simple shortcode that changes the color of a fragment of text red and makes it bold.

    1. In the file explorer, navigate to {project folder}/themes/{theme name}/layouts/shortcodes
    2. In the shortcodes folder, create a new file named color.html The name should be meaningful. It is used to call the shortcode in the Markdown files. It cannot contain any spaces.
    3. Open the file.
    4. In the file, add the following code: 
      <span style="color:red"><strong>{{ .Inner }}</strong></span>

      Explanation: The shortcode takes a fragment of plain text included in the code that called it and encases it with HTML markup.

    5. Save the file.
    6. Try using the shortcode in practice.
      Example: The shortcode is used to color the text you are reading now.

    For more information on creating shortcodes and the available functions and variables, refer to the Hugo documentation.

    Raw HTML

    You can include raw HTML markup in the Markdown files. This text is colored by using raw HTML.

    To achieve the same result as in the shortcode created earlier, you can do this:

    <span style="color:red"><strong>You can include raw HTML markup in the Markdown files. This text is colored by using raw HTML.</strong></span>

    NOTE: If the code does not work, you need to enable raw HTML in Hugo configuration.

    Enabling raw HTML

    1. In the file explorer, open the project folder.
    2. Open the config.toml file.
    3. Add the following lines:
    [markup]
    [markup.goldmark]
        [markup.goldmark.renderer]
            unsafe = true
