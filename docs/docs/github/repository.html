    Creating a repository

    1. Log in to GitHub.
    2. In the menu on the left, click New.
    3. In the Repository name field, enter a meaningful name.
      NOTE: If you host a project site from this repository, the name is part of the URL.
    4. Optional: In the Description field, enter a description.
    5. Make the repository public or private by selecting the corresponding radio button.
    6. Click Create repository.
      Result An empty repository is created.
    7. Open the Terminal by pressing CTRL + ALT + T.
    8. Navigate to the directory where you want to store the git repository. Example:
    marcin@marcin-HP:~$ cd Documents/gitRepositories
    1. Enter git clone git@github.com:{your GitHub username}/{your repository name}.git
    2. If prompted to add github.com to the list of known hosts, enter yes Result The repository is cloned into a new directory, named after the repository. You can now commit files from this location to GitHub.
